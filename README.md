## What is Learn.edodnan.com.au
Learn.edodnan.com.au is a branch of [Edodnan.com.au](https://edodnan.com.au) that focuses on being a place where all users can learn real things together, safely. We want everyone get acess to reliable information created by our writers and editors and fact-checked to that you are up-to-date on the coolest science facts. But we also teach things about math, computers,and we would love expanding this to something bigger.
